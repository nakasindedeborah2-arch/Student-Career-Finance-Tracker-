# Student-Career-Finance-Tracker-
Student Career & Finance Tracker is a Python application designed to help students manage both their career development and personal finances in one place.
The program allows users to track job applications, interview progress, networking contacts, income, expenses, savings goals, and overall career growth.

I developed this project to solve a real-world problem faced by many students: balancing career preparation with financial planning. The application stores and organizes user data, generates progress statistics, calculates savings trends, and provides insights into application success rates and financial goals.

While building this project, I strengthened my skills in Python programming, data structures, file management, problem-solving, and program design. One of the most rewarding parts of the project was designing a system that combines multiple features into a single, practical tool that students can use throughout their academic and professional journey.
